# Advanced Stock Portfolio Management System

A console-based portfolio management application built using C++ and Object-Oriented Programming.

## Features

- User Login & Registration
- Buy/Sell Stocks
- Mutual Funds
- Crypto Assets
- Portfolio Tracking
- Profit/Loss Analysis
- Market Simulation
- Transaction History
- File Handling

## Technologies

- C++
- OOP
- STL
- File Handling
